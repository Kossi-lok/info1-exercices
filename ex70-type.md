# Les types

## Ex 1
Quel est le type des expressions suivantes ?

Les variables suivantes sont déclarées:

```c
int val_i;
char c;
bool val_b;
double val_d;
```

Expression | Type
---|---
2 + 3 |
4.5 * val_i |
4 % 2 |
4 > 3 |
c + 4 |
c + val_d |
val_i > 3 && val_d < 0.0 |
